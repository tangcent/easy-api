easy-api


GitHub | Issues | Guide

A plugin that simplifies API development.
Help you generate API documentation from your code to postman/markdown

Usages
  • Open an existing Spring Controller file or select files and directories from the project navigation.
    Right-click on the file or directory and selecting 'EasyApi...' from the menu, or use the keyboard shortcuts 'Alt+Shift+E' (Windows) or 'Ctrl+E' (Mac)
